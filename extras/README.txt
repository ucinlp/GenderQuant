Instructions for downloading different datasets:

For Movie Summaries, do the following steps:
1. wget http://www.cs.cmu.edu/~ark/personas/data/MovieSummaries.tar.gz
2. tar xvzf MovieSummaries.tar.gz
3. python extras/processMovieDataset.py


For gutenberg novel dataset :
1. Run ./download.sh
2. python extras/novelTrainValTestSplitter.py

For news:
Please contact us so we can help you about it - it is not a free dataset.
